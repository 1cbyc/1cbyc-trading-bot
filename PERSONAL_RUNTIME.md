# running the bots

## Demo Testing (using launcher):
```bash
cd mt5-trading/demo-trading
py run-demo.py
```
## Demo Testing (direct run):
```bash
cd mt5-trading/demo-trading
py main.py
```

## eal Trading (Large Account):
```bash
cd mt5-trading/real-trading
py main-real.py
```

## Real Trading (Small Account):

```bash
cd mt5-trading/real-trading
py main-micro.py
```
